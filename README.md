# Sylvan

An x86-64 debugger

## Building from scratch

### Prerequisites

- Linux System (not sure if any linux specific things are used that aren't posix compliant)
- GCC compiler (GNU extensions are used)
- make

### Setup and Execution

#### Build

```make```

#### Run

```make run``` 
    
or

```./sylvan```

#### Clean

```make clean```
